# Create New AI Widget

This workflow guides you through the process of creating a fully customized AI widget for a client.

1.  **Run the `create-widget` skill**
    - This skill handles the entire process: discovery interview, configuration, code generation, and deployment.
    - View the skill instructions:
      // turbo
    - `view_file .agent/skills/create-widget/SKILL.md`

2.  **Follow the Skill Instructions**
    - The skill will guide you through:
      - technical verification
      - design discovery
      - feature configuration
      - code generation
      - knowledge base integration

3.  **Final Verification**
    - Ensure the widget is built and `script.js` is generated.
    - verify the info.json is created.
