# Quick Widget — Быстрое создание демо-виджета

## Зачем это нужно

Этот промпт позволяет максимально быстро создать демо-виджет для потенциального клиента.
Мы делаем виджет ДО того, как связываемся с клиентом — чтобы записать демо-видео,
показывающее, как виджет будет выглядеть на его сайте.

## Как использовать

1. Скопируй промпт ниже
2. Замени 3 поля на данные клиента
3. Вставь в AI-агента (Claude Code)
4. Дождись готового виджета (~3-5 минут)

---

## Промпт для AI-агента

```
/create-quick-widget

Создай быстрый демо-виджет для клиента:

- username: <КЛИЕНТ_ID>
- email: <EMAIL_КЛИЕНТА>
- website: <URL_САЙТА_КЛИЕНТА>
```

## Что делает агент автоматически

1. **Анализирует сайт** — заходит на сайт клиента, определяет цвета, шрифты, стиль, язык
2. **Собирает контент** — переходит по всем ссылкам, читает текст о компании, услугах, ценах, FAQ
3. **Создаёт виджет** — подбирает дизайн под стиль сайта, генерирует приветствие и быстрые ответы
4. **Заполняет базу знаний** — загружает собранную информацию в RAG для умных ответов
5. **Деплоит** — виджет доступен по ссылке для демонстрации

## Результат

После выполнения агент выдаст:

- Ссылку на демо-страницу: `http://localhost:3000/demo/dental?client=<client_id>`
- Embed-код для вставки на сайт: `<script src="https://domain/widgets/<client_id>/script.js"></script>`

## Дополнительные параметры (опционально)

Если нужно уточнить что-то конкретное, добавь к промпту:

```

- Язык виджета: русский / английский / украинский
- Тема: светлая / тёмная / авто
- Позиция: справа / слева
- Бизнес-контекст: <краткое описание, если сайт не информативный>
```
